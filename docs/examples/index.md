# 例题精解

基于国防科技大学2020-2023年共8套期末试卷（每年A/B卷）整理的典型例题，覆盖数字电子技术全部核心考点。

## 考试概况

| 年份/卷型 | 题型分布 | 总分 |
|:---|:---|:---:|
| 2020 A | 填空题(12分) + 选择题(20分) + 简答题(18分) + 综合题(50分) | 100 |
| 2020 B | 选择题(30分) + 填空题(20分) + 简答题(10分) + 综合题(40分) | 100 |
| 2021 A | 判断题(10分) + 填空题(15分) + 选择题(20分) + 分析简答(15分) + 综合题(40分) | 100 |
| 2021 B | 填空题(10分) + 选择题(20分) + 综合题(70分) | 100 |
| 2022 A | 填空题(15分) + 选择题(30分) + 综合题(55分) | 100 |
| 2022 B | 填空题(20分) + 选择题(30分) + 综合题(50分) | 100 |
| 2023 A | 判断题(10分) + 选择题(30分) + 综合题(60分) | 100 |
| 2023 B | 填空题(30分) + 选择题(30分) + 综合题(40分) | 100 |

## 例题导航

- [逻辑代数与组合设计](logic-algebra.md) — 对偶/反演、卡诺图化简（含无关项）、译码器与数据选择器实现逻辑函数
- [触发器波形分析](flipflop.md) — SR锁存器、JK触发器（含异步端）、单脉冲发生器
- [时序逻辑与计数器](sequential.md) — 三大方程分析法、74LS161任意进制计数器设计
- [DAC/ADC与波形发生器](dac-adc.md) — 输出电压计算、AD7520+ROM波形发生器
- [555定时器](pulse-555.md) — 施密特/多谐/单稳态三种电路、救护车扬声器
- [Verilog HDL](verilog.md) — 数据类型、计数器设计、代码功能识别

!!! tip "学习建议"
    建议按以下顺序学习：

    1. 先掌握逻辑代数化简（每年必考）
    2. 重点练习时序逻辑分析（三大方程法）
    3. 熟练计数器设计（反馈清零法/反馈置数法）
    4. 理解DAC/ADC计算公式
    5. 555定时器三种电路的参数计算
