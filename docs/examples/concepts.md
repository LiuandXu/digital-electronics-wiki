## 一、逻辑代数与门电路

### 1. 对偶式与反演式

| 项目 | 内容 |
|:---|:---|
| 来源 | 2020 A卷 · 填空题 |
| 考点 | 对偶规则、反演规则的运用 |
| 题目 | 逻辑式 \(Y=((AB'+C)'+D)'+A\) 的对偶式是____；反演式是____ |
| 关键结论 | 对偶：AND↔OR、0↔1 互换，变量不变；反演：还需取反变量 |
| 详解 | [逻辑代数 · 例题1](logic-algebra.md#例题1对偶式与反演式2020-a卷-填空题) |

### 2. 异或与同或运算性质

| 项目 | 内容 |
|:---|:---|
| 来源 | 2022 B卷 / 2023 B卷 · 填空3 |
| 考点 | 异或、同或运算在相同变量多次运算时的规律 |
| 题目 | 若 \(A \oplus \cdots \oplus A = 0\)，则 A 的个数为____数；若异或结果等于同或结果，则个数为____数 |
| 关键结论 | 偶数个相同变量异或 = 0，奇数个 = 变量本身；同或偶数个 = 1，奇数个 = 变量本身 |
| 详解 | [逻辑代数 · 例题6](logic-algebra.md#例题6异或运算性质2022-b卷-填空3-2023-b卷-填空3) |

### 3. TTL / CMOS 输入端接电阻判定

### 10. 倒 T 型 DAC 输出电压公式

| 项目 | 内容 |
|:---|:---|
| 来源 | 2023 B卷 · 填空10 |
| 考点 | 倒 T 型电阻网络 DAC 输出电压公式 |

## 概念题高频考点速记

| 考点 | 核心结论 | 出题次数 |
|:---|:---|:---:|
| TTL 输入端电阻/悬空等效 | 悬空="1"，\(R{<}0.7k\Omega\)="0"，\(R{>}2k\Omega\)="1" | 3 |
| CMOS 输入端处理 | 接电阻="0"，悬空禁止 | 2 |
| CMOS 电路识别 | PMOS 串+NMOS 并→NAND；PMOS 并+NMOS 串→NOR | 1 |
| 对偶 / 反演规则 | 对偶不取反，反演取反 | 1 |
| 异或 / 同或性质 | 偶数个异或=0，奇数个=本身 | 1 |
| DAC 输出电压公式 | \(V_O = -V_{REF} \cdot D / 2^n\) | 1 |
| 权电阻规律 | \(R_i = R \times 2^{n-1-i}\) | 1 |
| ADC 类型对比 | 并联最快、双积分抗干扰最强、逐次渐近最常用 | 1 |
| 555 三种电路识别 | 施密特/多谐/单稳态的引脚连接特征 | 1 |
| Verilog 数据类型 | 默认 wire，reg 用于 always 块 | 1 |
| Verilog 标识符规则 | 不能以数字开头 | 1 |
| 代码功能识别 | 从表达式判断半加器/半减器等 | 1 |
