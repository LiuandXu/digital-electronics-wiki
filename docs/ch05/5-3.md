# 5.3 寄存器与移位寄存器

寄存器是数字系统中用于暂存二进制数据的常用时序逻辑器件。由多个触发器构成的寄存器不仅能存储数据，还可以通过适当的连线实现数据的移位、并串转换等功能。

---

## 一、寄存器概述

### 1.1 定义与分类

**寄存器**（Register）是由一组触发器构成的、用于存储一组二进制代码的时序逻辑电路。根据功能不同，寄存器分为：

- **基本寄存器（数码寄存器）**：仅具有数据寄存功能，可并行存入、并行取出
- **移位寄存器**：除寄存功能外，还能在时钟脉冲作用下使数据逐位左移或右移

### 1.2 n 位寄存器

n 个触发器可以构成 n 位寄存器，能存储 n 位二进制信息。通常采用 D 触发器构成，因为 D 触发器的特性方程 \(Q^{n+1} = D\) 最为简洁。

---

## 二、基本数码寄存器

### 2.1 结构原理

基本数码寄存器由 n 个 D 触发器并联构成，所有触发器共用同一个时钟 CP。当 CP 上升沿（或下降沿）到来时，n 位输入数据同时存入对应的触发器中。

**工作方式**：并行输入 \(\to\) 并行输出

### 2.2 四位寄存器示例

使用 4 个 D 触发器构成 4 位寄存器（74LS175 为典型 TTL 四 D 触发器芯片）：

- 输入端：\(D_3, D_2, D_1, D_0\)
- 输出端：\(Q_3, Q_2, Q_1, Q_0\)
- 时钟端：CP（公共时钟）
- 清零端：\(\overline{CR}\)（公共异步清零）

当时钟上升沿到达时，\(Q_i^{n+1} = D_i\)（\(i = 0, 1, 2, 3\)），四位数据被同时存入。

---

## 三、移位寄存器

### 3.1 基本结构

移位寄存器将各触发器的输出端与相邻触发器的输入端依次级联，在时钟脉冲作用下实现数据的逐位移动。

**基本连接方式**：前一触发器的输出 \(\to\) 后一触发器的输入

### 3.2 单向移位寄存器

#### （1）右移移位寄存器

数据从最左边触发器输入，逐位向右移动：

- 连接方式：\(D_i = Q_{i-1}\)（当前触发器输入 = 相邻左边触发器的输出）
- 最左端 \(D_0\) 接外部串行输入

工作过程（以 4 位为例）：

| CP 序号 | \(Q_0\) | \(Q_1\) | \(Q_2\) | \(Q_3\) |
|:---:|:---:|:---:|:---:|:---:|
| 初态 | 0 | 0 | 0 | 0 |
| 1 | \(D_{in}\) | 0 | 0 | 0 |
| 2 | -- | \(D_{in}\) | 0 | 0 |
| 3 | -- | -- | \(D_{in}\) | 0 |
| 4 | -- | -- | -- | \(D_{in}\) |

经 n 个 CP 后，串行输入的 n 位数据依次存入 n 个触发器中。此后，可从输出端并行读取全部数据（**串入并出**），或继续移动从最右端逐个串行输出（**串入串出**）。

#### （2）左移移位寄存器

数据从最右边触发器输入，逐位向左移动：

- 连接方式：\(D_i = Q_{i+1}\)（当前触发器输入 = 相邻右边触发器的输出）

### 3.3 双向移位寄存器

通过增加方向控制逻辑，可以在同一电路中实现左移和右移功能。

典型芯片 **74LS194**（4 位双向移位寄存器）的功能：

| 工作模式 | 功能 | 实现方式 |
|:---:|:---|:---|
| 保持 | 状态不变 | CP 被禁止 |
| 右移 | \(Q_i \to Q_{i+1}\) | 数据从 \(D_{SR}\)（右移串入端）输入 |
| 左移 | \(Q_i \to Q_{i-1}\) | 数据从 \(D_{SL}\)（左移串入端）输入 |
| 并行置数 | 同步置入 4 位数据 | \(D_0 \sim D_3\) 并行输入 |

---

## 四、移位寄存器的应用

### 4.1 数据格式转换

移位寄存器最核心的应用是实现**串行-并行转换**和**并行-串行转换**：

- **串行转并行**：将串行输入的数据经移位后在并行输出端同时读取
- **并行转串行**：将并行数据通过置数功能存入，然后逐位串行输出

这在需要长距离传输数据的通信系统中极为重要（节省传输线路，一条数据线即可）。

### 4.2 序列信号发生器

利用移位寄存器和反馈逻辑，可以产生特定的周期性序列信号（伪随机序列、m 序列等）。反馈函数 \(F(Q_0, Q_1, \ldots, Q_{n-1})\) 接入移位寄存器输入端，形成闭环反馈。

### 4.3 环形计数器

将移位寄存器的末级输出反馈到首级输入（无需外加数据输入），即构成环形计数器。

n 位环形计数器有 n 个有效状态，需要预置一个初始状态（通常为 1000 或 0001）后才能开始循环。状态循环举例（4 位，初态 1000）：

\[
1000 \to 0100 \to 0010 \to 0001 \to 1000 \to \cdots
\]

### 4.4 扭环形计数器（约翰逊计数器）

将移位寄存器的末级反相输出 \(\overline{Q_{n-1}}\) 反馈到首级输入 \(D_0\)，构成扭环形计数器。

n 位扭环形计数器有 \(2n\) 个有效状态。以 4 位为例，状态循环为：

\[
0000 \to 1000 \to 1100 \to 1110 \to 1111 \to 0111 \to 0011 \to 0001 \to 0000 \to \cdots
\]

共 8 个有效状态（2n = 8）。扭环形计数器能用较少的触发器获得较多有效状态，但需要额外的译码电路来识别各状态。

## 五、寄存器应用扩展

### 5.1 多级寄存器级联

将多个移位寄存器芯片级联（前级末位输出接后级首位输入），可以扩展寄存器的位宽。例如，两片 4 位移位寄存器级联可构成 8 位移位寄存器。

### 5.2 数据锁存与缓冲

寄存器（或锁存器）在数字系统中广泛用作数据缓冲器，例如：

- **总线驱动器**：在 CPU 与外部设备间进行数据暂存和缓冲
- **流水线寄存器**：在流水线结构的处理器中，各级之间插入寄存器暂存中间结果
- **端口寄存器**：微控制器中 GPIO 端口的数据寄存器和方向寄存器

## 七、寄存器设计示例

### 示例：设计一个 3 位双向移位寄存器

**要求**：当控制信号 M = 0 时右移，M = 1 时左移。

**设计方案**：

采用 3 个 D 触发器，通过二选一数据选择器实现方向控制：

- 对于第 i 个触发器（i = 0, 1, 2）：
  - \(D_i = M \cdot Q_{i+1} + \overline{M} \cdot Q_{i-1}\)（当 M = 0 时右移，\(D_i = Q_{i-1}\)；当 M = 1 时左移，\(D_i = Q_{i+1}\)）

端触发器的特殊处理：
- 最左端：\(D_0 = M \cdot Q_1 + \overline{M} \cdot D_{SR}\)（\(D_{SR}\) 为右移串行输入）
- 最右端：\(D_2 = M \cdot D_{SL} + \overline{M} \cdot Q_1\)（\(D_{SL}\) 为左移串行输入）

## 八、本节小结

| 寄存器类型 | 输入方式 | 输出方式 | 典型应用 |
|:---:|:---:|:---:|:---|
| 数码寄存器 | 并行 | 并行 | 数据暂存、缓冲 |
| 右移寄存器 | 串行 | 串行/并行 | 串并转换 |
| 左移寄存器 | 串行 | 串行/并行 | 串并转换 |
| 双向移位寄存器 | 串行/并行 | 串行/并行 | 通用数据移位 |
| 环形计数器 | 反馈 | 并行 | 节拍发生器 |
| 扭环形计数器 | 反馈 | 并行 | 状态序列发生 |

!!! warning "易错点"
    1. 移位寄存器中每来一个 CP，数据只移动一位--不能一次移动多位
    2. 串入并出移位寄存器需要 n 个 CP 才能将 n 位数据全部移入，然后才能并行读取
    3. 环形计数器的初始状态必须通过预置功能设置，否则可能停在无效状态
    4. 移位寄存器和基本数码寄存器的核心区别在于触发器之间是否有级联（移位连接）
