# 5.4 计数器

计数器是数字系统中使用最广泛的时序逻辑器件之一，其基本功能是对输入的时钟脉冲个数进行计数。计数器不仅用于计数，还可用于分频、定时、产生节拍脉冲和序列信号等。

---

## 一、计数器概述

### 1.1 计数器的分类

| 分类标准 | 类型 | 说明 |
|:---:|:---|:---|
| 计数方式 | **加法计数器** | 每来一个 CP，计数加 1 |
| 计数方式 | **减法计数器** | 每来一个 CP，计数减 1 |
| 计数方式 | **可逆计数器** | 可加可减，由控制信号决定 |
| 时钟控制 | **同步计数器** | 所有触发器共用同一时钟 |
| 时钟控制 | **异步计数器** | 各触发器时钟不同，逐级传递 |
| 计数进制 | **二进制计数器** | 模为 \(2^n\) |
| 计数进制 | **十进制计数器** | 模为 10 |
| 计数进制 | **任意进制计数器** | 模为任意整数 N |

### 1.2 基本计数单元 -- T' 触发器

T' 触发器的特性方程为 \(Q^{n+1} = \overline{Q^n}\)，每来一个 CP 脉冲状态翻转一次，因此它是构成计数器最基本、最核心的单元。n 个 T' 触发器级联可构成 n 位二进制计数器。

---

## 二、同步计数器

### 2.1 同步二进制加法计数器

**特点**：所有触发器的 CP 端接在一起，共用同一个时钟信号（同步）。触发器的翻转受控制逻辑（由低位触发器的输出组合逻辑决定）控制。

**设计原理**（以 JK 触发器构成的 3 位同步二进制加法计数器为例）：

第一个触发器 FF0：每来一个 CP 翻转一次，故 \(J_0 = K_0 = 1\)（T' 触发器模式）。

第二个触发器 FF1：在 \(Q_0 = 1\) 时，下一个 CP 使 FF1 翻转。故 \(J_1 = K_1 = Q_0^n\)。

第三个触发器 FF2：在 \(Q_1 = Q_0 = 1\) 时，下一个 CP 使 FF2 翻转。故 \(J_2 = K_2 = Q_1^n \cdot Q_0^n\)。

**通用递推公式（n 位同步二进制加法计数器）**：

\[
J_i = K_i = Q_0^n \cdot Q_1^n \cdot \ldots \cdot Q_{i-1}^n \quad (i = 1, 2, \ldots, n-1)
\]

其中 \(J_0 = K_0 = 1\)。

**状态转换（3 位，共 8 个状态）**：

\[
000 \to 001 \to 010 \to 011 \to 100 \to 101 \to 110 \to 111 \to 000 \to \cdots
\]

### 2.2 同步二进制减法计数器

减法计数器每来一个 CP，计数值减 1。

**设计原理**（3 位，JK 触发器）：

- FF0：\(J_0 = K_0 = 1\)
- FF1：在 \(Q_0 = 0\) 时翻转（注意：减 1 时低位从 0 变 1，向高位借位）。故 \(J_1 = K_1 = \overline{Q_0^n}\)
- FF2：\(J_2 = K_2 = \overline{Q_1^n} \cdot \overline{Q_0^n}\)

### 2.3 同步十进制加法计数器（8421BCD 码）

模 10 计数器有 10 个有效状态（0000 \(\sim\) 1001），从 1001 加 1 后应回到 0000（而非 1010）。

**设计思路**：在二进制计数器基础上，当计数到 1001 时，通过反馈逻辑使计数器在下一个 CP 归零。

---

## 三、异步计数器

### 3.1 异步二进制加法计数器

**结构特点**：各触发器的时钟端不接在一起，低位触发器的输出作为高位触发器的时钟。

以 4 个下降沿触发的 JK 触发器构成的 4 位异步二进制加法计数器为例：

- \(CP_0 = CP\)（外部时钟）
- \(CP_1 = Q_0\)（FF0 的输出作为 FF1 的时钟）
- \(CP_2 = Q_1\)
- \(CP_3 = Q_2\)

各触发器以 T' 方式工作（\(J_i = K_i = 1\)）。

### 3.2 异步二进制减法计数器

与加法计数器的区别在于：高位触发器的时钟取自低位的 \(\overline{Q}\) 端（而非 Q 端）。

### 3.3 同步与异步计数器对比

| 特性 | 同步计数器 | 异步计数器 |
|:---:|:---|:---|
| 时钟连接 | 所有 CP 并联 | 低位 Q 输出接高位 CP |
| 状态翻转 | 同时翻转 | 逐级翻转（有延迟） |
| 工作速度 | 快 | 慢 |
| 电路复杂度 | 较复杂（需控制逻辑） | 简单 |
| 毛刺风险 | 低 | 高（过渡状态多） |
| 典型芯片 | 74LS161, 74LS163 | 74LS93, 74LS393 |
| 分析难度 | 一致 | 需逐级判断时钟有效性 |

## 四、任意进制计数器

### 4.1 问题引入

实际应用中常需要非 \(2^n\) 进制的计数器（如六进制、十进制等）。设计任意进制计数器有两种基本方法：**反馈清零法**和**反馈置数法**。

### 4.2 反馈清零法

**原理**：利用计数器芯片的异步清零端 \(\overline{CR}\)，当计数达到目标模数 N 的状态时，通过组合逻辑产生清零信号，使计数器立即回到 0 状态。

**设计步骤**：

1. 选择进制数大于 N 的计数器芯片（如 N 进制用 74LS161/163 等模 16 芯片）
2. 写出模 N 对应的二进制状态码
3. 将状态码中为 1 的输出位接入与非门，产生清零脉冲
4. 将与非门输出接至 \(\overline{CR}\) 端

**例题**：用 74LS161（4 位同步二进制计数器，异步清零）构成模 6 计数器。

解：
- 模 6 计数序列：0000 \(\to\) 0001 \(\to\) 0010 \(\to\) 0011 \(\to\) 0100 \(\to\) 0101
- 当计数到 0110（6）时，\(Q_2 = Q_1 = 1\)，用与非门产生清零信号
- \(\overline{CR} = \overline{Q_2 \cdot Q_1}\)
- 注意：0110 是过渡状态（瞬间出现即清零），有效计数状态为 0000 \(\sim\) 0101

!!! warning "易错点"
    反馈清零法中，清零信号对应的状态（如上述 0110）是一个瞬态，不计入有效计数循环。模 N 计数器共有 N 个有效状态。如果清零是异步的，过渡状态持续极短时间；如果清零是同步的（如 74LS163），需要等下一个 CP 才会清零，要特别注意状态安排。

### 4.3 反馈置数法

**原理**：利用计数器芯片的同步置数功能，当计数到某个状态时，在下一个 CP 将计数器置为预设值，从而实现任意进制计数。

**设计步骤**：

1. 确定 N 进制计数的 M 个状态（M = N）
2. 在序号为 N-1 的状态时产生置数信号
3. 将置入数据设置为计数循环的起始值

**例题**：用 74LS161（同步置数）构成模 6 计数器（计数序列：0011 \(\sim\) 1000）。

解：
- 6 个有效状态：0011 \(\to\) 0100 \(\to\) 0101 \(\to\) 0110 \(\to\) 0111 \(\to\) 1000
- 计数到 1000 时产生置数信号：\(\overline{LD} = \overline{Q_3 \cdot \overline{Q_2} \cdot \overline{Q_1} \cdot \overline{Q_0}}\)
- 置入数据：\(D_3 D_2 D_1 D_0 = 0011\)
- 1000 \(\to\)（置数）\(\to\) 0011，形成闭环循环

### 4.4 级联扩展

当所需计数器模数超过单片计数器的最大模数时，可将多片计数器级联：

- **同步级联**：各片共用同一 CP，低位片的进位/借位输出控制高位片的使能端
- **异步级联**：低位片的进位/借位输出作为高位片的 CP

同步级联速度更快，异步级联连线更简单。以两片 74LS161 级联构成 8 位（模 256）计数器为典型应用。

### 5.1 74LS161 功能详解

74LS161 是 4 位同步二进制计数器，各引脚功能：

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(\overline{CR}\) | 异步清零 | 低电平有效，立刻清零 |
| \(\overline{LD}\) | 同步置数 | 低电平有效，下一 CP 置数 |
| \(EP, ET\) | 计数使能 | 均为高电平时计数 |
| CP | 时钟 | 上升沿触发 |
| \(D_0 \sim D_3\) | 并行数据输入 | 置数时载入的数据 |
| \(Q_0 \sim Q_3\) | 计数输出 | 当前计数值 |
| \(RCO\) | 进位输出 | 计数到 1111 且 ET = 1 时输出高电平 |

## 六、计数器设计综合例题

### 例题 1：分析异步十进制计数器

**电路**：4 个下降沿 JK 触发器构成的异步计数器，FF3 的时钟来自 \(Q_0\)。

要求：分析计数循环并确定模数。

**解**：

（1）确定各触发器工作方式：各 JK 端均悬空（视为 \(J = K = 1\)，即 T' 模式）

（2）确定各触发器时钟：
- \(CP_0 = CP\)（外部时钟）
- \(CP_1 = Q_0\)（FF1 在 \(Q_0\) 下降沿触发）
- \(CP_2 = Q_1\)
- \(CP_3 = Q_0\)（注意 FF3 的时钟也来自 \(Q_0\)）

（3）列状态转换表，从 0000 开始：
- 0000 \(\to\) 0001 \(\to\) 0010 \(\to\) 0011 \(\to\) 0100 \(\to\) 0101 \(\to\) 0110 \(\to\) 0111 \(\to\) 1000 \(\to\) 1001 \(\to\) 0000

（4）结论：具有自启动能力的**异步十进制（模 10）加法计数器**。

### 例题 2：设计一个同步六进制加法计数器

**要求**：用 JK 触发器设计，具有自启动能力。

**解**：

**步骤一：逻辑抽象**

需要 6 个有效状态，最少需要 3 个触发器（\(2^2 < 6 \le 2^3\)）。

状态编码：000 \(\to\) 001 \(\to\) 010 \(\to\) 011 \(\to\) 100 \(\to\) 101 \(\to\) 000

输出：计数到 101 时产生进位输出 C = 1。

**步骤二：求状态方程**

根据状态转换序列，画出各触发器的卡诺图，化简得状态方程。

**步骤三：确定驱动方程**

将状态方程与 JK 触发器的特性方程对比（\(Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n\)），确定各 J、K 端的表达式。

**步骤四：检查自启动**

未使用的状态（110, 111）能否在有限 CP 内进入有效循环？若能，则具有自启动能力；若不能，则需要修改电路设计。

## 七、本节重点总结

| 知识点 | 核心内容 |
|:---|:---|
| 同步 vs 异步 | 时钟连接方式不同，同步全部时钟并联；异步低位输出驱动高位时钟 |
| 反馈清零法 | 计数到 N 时产生清零信号，适用于异步清零芯片（如 74LS161），N 对应的状态是瞬态 |
| 反馈置数法 | 计数到终止值时产生置数信号，适用于同步置数芯片，计数器在下一个 CP 加载预设值 |
| 级联扩展 | 将多片计数器级联可实现更大模数，同步级联优于异步级联 |
| 自启动检查 | 设计完成后必须检查所有无效状态是否能在有限 CP 内进入有效循环 |
| 时序图 | 同步计数器各输出同时变化；异步计数器逐级延迟，存在过渡毛刺 |

!!! warning "易错点"
    1. **异步计数器分析时**：必须逐一判断各触发器是否有有效时钟沿到来。无时钟沿 \(\to\) 保持现态；有时钟沿 \(\to\) 按状态方程计算次态
    2. **反馈清零法**：清零状态为瞬态（异步清零）或维持一个 CP（同步清零）。把握不清的话可以直接从状态转换图判断模数
    3. **同步清零 vs 异步清零**：74LS161 是异步清零（立即清零），74LS163 是同步清零（下一个 CP 才清零）。设计 N 进制计数器时，清零/置数信号的触发状态不同
    4. **级联时**：注意进位信号的产生条件以及芯片使能端的连接方式
    5. **自启动**：如果无效状态形成孤立循环，电路无法自启动，需要修改设计（增加引导逻辑或使用置数/清零）
