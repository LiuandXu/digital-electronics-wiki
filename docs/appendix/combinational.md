# 附录 A：常见组合逻辑电路芯片

本附录汇总常用的组合逻辑集成电路芯片，包括编码器、译码器、数据选择器、加法器和数值比较器等。

---

## A.1 74148 — 8 线-3 线优先编码器

### 功能概述

74148 是 8 线-3 线优先编码器，将 8 条输入线编码为 3 位二进制输出。输入和输出均为**低电平有效**，支持芯片级联扩展。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(I_0 \sim I_7\) | 编码输入 | 低电平有效，\(I_7\) 优先级最高 |
| \(A_2, A_1, A_0\) | 编码输出 | 3 位二进制码（反码输出） |
| \(EI\) | 使能输入 | 低电平有效，允许编码 |
| \(EO\) | 使能输出 | 低电平有效，表示无输入且已使能 |
| \(GS\) | 组信号输出 | 低电平有效，表示有有效编码输入 |

### 功能表

### 典型应用

- **级联扩展**：利用 \(EI\)、\(EO\)、\(GS\) 可将两片 74148 级联为 16 线-4 线编码器
- **键盘编码**：将 8 键输入编码为 3 位二进制码

## A.2 74138 — 3 线-8 线译码器

### 功能概述

74138 是 3 线-8 线二进制译码器，将 3 位二进制输入译码为 8 条输出线之一。输出为**低电平有效**，具有 3 个使能端（1 个高有效 + 2 个低有效），便于级联。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(A_2, A_1, A_0\) | 地址输入 | 3 位二进制输入 |
| \(Y_0 \sim Y_7\) | 译码输出 | 低电平有效，对应输入的译码输出 |
| \(G_1\) | 使能输入 | 高电平有效 |
| \(\overline{G_{2A}}, \overline{G_{2B}}\) | 使能输入 | 低电平有效 |

### 功能表

| \(G_1\) | \(\overline{G_{2A}}\) | \(\overline{G_{2B}}\) | \(A_2\) | \(A_1\) | \(A_0\) | 输出 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---|
| 0 | X | X | X | X | X | 全 1 |
| X | 1 | X | X | X | X | 全 1 |
| X | X | 1 | X | X | X | 全 1 |
| 1 | 0 | 0 | 0 | 0 | 0 | \(Y_0 = 0\)，其余为 1 |
| 1 | 0 | 0 | 0 | 0 | 1 | \(Y_1 = 0\)，其余为 1 |
| 1 | 0 | 0 | 0 | 1 | 0 | \(Y_2 = 0\)，其余为 1 |
| 1 | 0 | 0 | 0 | 1 | 1 | \(Y_3 = 0\)，其余为 1 |
| 1 | 0 | 0 | 1 | 0 | 0 | \(Y_4 = 0\)，其余为 1 |
| 1 | 0 | 0 | 1 | 0 | 1 | \(Y_5 = 0\)，其余为 1 |
| 1 | 0 | 0 | 1 | 1 | 0 | \(Y_6 = 0\)，其余为 1 |
| 1 | 0 | 0 | 1 | 1 | 1 | \(Y_7 = 0\)，其余为 1 |

### 典型应用

- **存储器地址译码**：将高位地址译码为片选信号
- **数据分配器**：将使能端作为数据输入，地址端作为选择信号
- **实现组合逻辑**：利用译码输出 + 与非门实现任意逻辑函数

## A.3 7448 — BCD-七段译码器

### 功能概述

7448 将 4 位 BCD 码译码为七段显示信号，驱动**共阴极** LED 数码管，输出为高电平有效。内置上拉电阻，无需外接限流电阻即可直接驱动数码管。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(A_3 \sim A_0\) | BCD 输入 | 4 位 BCD 码（0000~1001 对应 0~9） |
| \(Y_a \sim Y_g\) | 段输出 | 七段显示信号，高电平有效 |
| \(\overline{LT}\) | 灯测试 | 低电平时所有段全亮，显示"8" |
| \(\overline{RBI}\) | 动态灭零输入 | 低电平时将前导零消隐 |
| \(\overline{BI}/\overline{RBO}\) | 消隐输入/灭零输出 | 双向引脚，输入低强制消隐；输出低表示正在灭零 |

### 显示对应表

| BCD 输入 | 显示 | BCD 输入 | 显示 |
|:---:|:---:|:---:|:---:|
| 0000 | 0 | 0101 | 5 |
| 0001 | 1 | 0110 | 6 |
| 0010 | 2 | 0111 | 7 |
| 0011 | 3 | 1000 | 8 |
| 0100 | 4 | 1001 | 9 |

> 1010~1111 为无效 BCD 码，输出取决于具体实现（通常消隐或显示特殊符号）。

### 典型应用

- **数码管显示**：驱动共阴极 LED 数码管显示十进制数字
- **动态灭零**：利用 \(\overline{RBI}\) 和 \(\overline{RBO}\) 实现多位显示中前导零消隐

## A.4 74151 — 8 选 1 数据选择器

### 功能概述

74151 是 8 选 1 数据选择器（Multiplexer），根据 3 位地址选择 8 路输入中的一路输出，提供原码（\(Y\)）和反码（\(\overline{Y}\)）双输出。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(D_0 \sim D_7\) | 数据输入 | 8 路数据输入 |
| \(A_2, A_1, A_0\) | 地址输入 | 3 位选择信号 |
| \(\overline{EN}\) | 使能输入 | 低电平有效 |
| \(Y\) | 原码输出 | 选中数据原值 |
| \(\overline{Y}\) | 反码输出 | 选中数据的反值 |

### 功能表

| \(\overline{EN}\) | \(A_2\) | \(A_1\) | \(A_0\) | \(Y\) | \(\overline{Y}\) |
|:---:|:---:|:---:|:---:|:---:|:---:|
| 1 | X | X | X | 0 | 1 |
| 0 | 0 | 0 | 0 | \(D_0\) | \(\overline{D_0}\) |
| 0 | 0 | 0 | 1 | \(D_1\) | \(\overline{D_1}\) |
| 0 | 0 | 1 | 0 | \(D_2\) | \(\overline{D_2}\) |
| 0 | 0 | 1 | 1 | \(D_3\) | \(\overline{D_3}\) |
| 0 | 1 | 0 | 0 | \(D_4\) | \(\overline{D_4}\) |
| 0 | 1 | 0 | 1 | \(D_5\) | \(\overline{D_5}\) |
| 0 | 1 | 1 | 0 | \(D_6\) | \(\overline{D_6}\) |
| 0 | 1 | 1 | 1 | \(D_7\) | \(\overline{D_7}\) |

### 典型应用

- **实现组合逻辑函数**：将逻辑函数接为数据输入，地址端作为逻辑变量
- **数据路由**：在多路数据中选择一路输出
- **并行转串行**：配合计数器实现并-串转换

## A.5 74283 — 4 位二进制超前进位加法器

### 功能概述

74283 是 4 位二进制全加器，采用超前进位（Carry Look-Ahead）结构，内部产生各级进位，避免了行波进位的延迟，运算速度快。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(A_3 \sim A_0\) | 被加数 | 4 位二进制输入 A |
| \(B_3 \sim B_0\) | 加数 | 4 位二进制输入 B |
| \(C_0\) | 进位输入 | 最低位进位输入 |
| \(S_3 \sim S_0\) | 和输出 | 4 位二进制和 |
| \(C_4\) | 进位输出 | 最高位进位输出 |

### 功能说明

\[
S = A + B + C_0
\]

- 当 \(A + B + C_0 \leq 15\) 时，\(C_4 = 0\)，结果为 \(S_3 S_2 S_1 S_0\)
- 当 \(A + B + C_0 \geq 16\) 时，\(C_4 = 1\)，结果为 \(C_4 S_3 S_2 S_1 S_0\)（5 位）

### 典型应用

- **多位加法器扩展**：利用 \(C_4\) 接下一片的 \(C_0\) 级联为 8/12/16 位加法器
- **减法运算**：配合补码实现减法（B 取反 + 1）
- **BCD 加法**：结合修正电路实现 BCD 码加法

## A.6 7485 — 4 位数值比较器

### 功能概述

7485 对两个 4 位二进制数进行大小比较，输出"大于"、"等于"或"小于"三种结果。具有级联输入端，可扩展为更多位数的比较器。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(A_3 \sim A_0\) | 数 A 输入 | 4 位二进制数 A |
| \(B_3 \sim B_0\) | 数 B 输入 | 4 位二进制数 B |
| \(a \gt b\) | 级联输入 | 高电平表示上级 A 大于 B |
| \(a = b\) | 级联输入 | 高电平表示上级 A 等于 B |
| \(a \lt b\) | 级联输入 | 高电平表示上级 A 小于 B |
| \(A \gt B\) | 输出 | 高电平表示 A 大于 B |
| \(A = B\) | 输出 | 高电平表示 A 等于 B |
| \(A \lt B\) | 输出 | 高电平表示 A 小于 B |

### 功能表（简表）

| 比较 | \(A \gt B\) | \(A = B\) | \(A \lt B\) |
|:---:|:---:|:---:|:---:|
| \(A_3 \gt B_3\) | 1 | 0 | 0 |
| \(A_3 \lt B_3\) | 0 | 0 | 1 |
| \(A_3 = B_3\)，比较 \(A_2, B_2\) | — | — | — |
| \(A = B\)（全部相等），\(a \gt b = 1\) | 1 | 0 | 0 |
| \(A = B\)（全部相等），\(a = b = 1\) | 0 | 1 | 0 |
| \(A = B\)（全部相等），\(a \lt b = 1\) | 0 | 0 | 1 |

### 典型应用

- **多位比较**：级联多片 7485 实现 8/12/16 位数值比较
- **排序电路**：配合多路选择器实现数字排序
- **地址比较**：在存储器系统中比较地址范围
