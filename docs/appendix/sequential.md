# 附录 B：常见时序逻辑电路芯片

本附录汇总常用的时序逻辑集成电路芯片，包括触发器、寄存器和计数器等。

---

## B.1 74LS74 — 双 D 触发器

### 功能概述

74LS74 内含两个独立的 D 触发器，上升沿触发，每个触发器带有独立的置位（\(\overline{S_D}\)）和复位（\(\overline{R_D}\)）端，低电平有效。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP_1, CP_2\) | 时钟输入 | 上升沿触发 |
| \(D_1, D_2\) | 数据输入 | 触发前的数据输入 |
| \(Q_1, Q_2\) | 原码输出 | 触发器输出 |
| \(\overline{Q_1}, \overline{Q_2}\) | 反码输出 | 触发器反相输出 |
| \(\overline{S_{D1}}, \overline{S_{D2}}\) | 异步置位 | 低电平有效，强制 Q = 1 |
| \(\overline{R_{D1}}, \overline{R_{D2}}\) | 异步复位 | 低电平有效，强制 Q = 0 |

### 功能表

| \(\overline{R_D}\) | \(\overline{S_D}\) | \(CP\) | \(D\) | \(Q^{n+1}\) |
|:---:|:---:|:---:|:---:|:---:|
| 0 | 1 | X | X | 0（异步复位） |
| 1 | 0 | X | X | 1（异步置位） |
| 1 | 1 | \(\uparrow\) | 0 | 0 |
| 1 | 1 | \(\uparrow\) | 1 | 1 |
| 1 | 1 | 0 / 1 / \(\downarrow\) | X | \(Q^n\)（保持） |

> \(\overline{R_D}\) 和 \(\overline{S_D}\) 不允许同时为 0。

### 特性方程

\[
Q^{n+1} = D
\]

### 典型应用

- **寄存器**：多片 74LS74 并联构成 D 触发器阵列
- **分频器**：D 触发器接成 T' 模式（\(\overline{Q}\) 接 D），实现 2 分频
- **同步电路**：作为同步时序电路的基本存储元件

## B.2 74LS112 — 双 JK 触发器

### 功能概述

74LS112 内含两个独立的 JK 触发器，下降沿触发，带有独立的异步置位和复位端。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP_1, CP_2\) | 时钟输入 | 下降沿触发 |
| \(J_1, J_2\) | J 输入 | 置位控制端 |
| \(K_1, K_2\) | K 输入 | 复位控制端 |
| \(Q_1, Q_2\) | 原码输出 | 触发器输出 |
| \(\overline{Q_1}, \overline{Q_2}\) | 反码输出 | 触发器反相输出 |
| \(\overline{S_{D1}}, \overline{S_{D2}}\) | 异步置位 | 低电平有效 |
| \(\overline{R_{D1}}, \overline{R_{D2}}\) | 异步复位 | 低电平有效 |

### 功能表

| \(\overline{R_D}\) | \(\overline{S_D}\) | \(CP\) | \(J\) | \(K\) | \(Q^{n+1}\) | 功能 |
|:---:|:---:|:---:|:---:|:---:|:---:|:---|
| 0 | 1 | X | X | X | 0 | 异步复位 |
| 1 | 0 | X | X | X | 1 | 异步置位 |
| 1 | 1 | \(\downarrow\) | 0 | 0 | \(Q^n\) | 保持 |
| 1 | 1 | \(\downarrow\) | 0 | 1 | 0 | 复位 |
| 1 | 1 | \(\downarrow\) | 1 | 0 | 1 | 置位 |
| 1 | 1 | \(\downarrow\) | 1 | 1 | \(\overline{Q^n}\) | 翻转 |
| 1 | 1 | 其他 | X | X | \(Q^n\) | 保持 |

### 特性方程

\[
Q^{n+1} = J\overline{Q^n} + \overline{K}Q^n
\]

### 典型应用

- **计数器**：JK 触发器是构成同步/异步计数器的核心元件
- **分频器**：J = K = 1 时实现 T' 模式，2 分频
- **移位寄存器**：配合逻辑门构成移位寄存器

## B.3 74LS175 — 四 D 触发器

### 功能概述

74LS175 内含 4 个 D 触发器，共享同一时钟和异步清零端，适合用作并行数据寄存。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP\) | 时钟输入 | 上升沿触发，4 个触发器共享 |
| \(\overline{CR}\) | 异步清零 | 低电平有效，强制全部输出为 0 |
| \(D_0 \sim D_3\) | 数据输入 | 4 路并行数据输入 |
| \(Q_0 \sim Q_3\) | 原码输出 | 4 路原码输出 |
| \(\overline{Q_0} \sim \overline{Q_3}\) | 反码输出 | 4 路反码输出 |

### 功能表

| \(\overline{CR}\) | \(CP\) | \(D\) | \(Q^{n+1}\) |
|:---:|:---:|:---:|:---:|
| 0 | X | X | 0 |
| 1 | \(\uparrow\) | 0 | 0 |
| 1 | \(\uparrow\) | 1 | 1 |
| 1 | 其他 | X | \(Q^n\) |

### 典型应用

- **数据寄存器**：暂存 4 位并行数据
- **同步输出缓冲**：将异步数据同步到时钟域

---

## B.4 74LS194 — 4 位双向移位寄存器

### 功能概述

74LS194 是 4 位双向移位寄存器，支持左移、右移、并行置数和保持四种工作模式，由 \(S_1, S_0\) 两位控制信号选择。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP\) | 时钟输入 | 上升沿触发 |
| \(\overline{CR}\) | 异步清零 | 低电平有效 |
| \(S_1, S_0\) | 模式控制 | 选择工作模式 |
| \(D_R\) | 右移串行输入 | 右移时数据从此输入 |
| \(D_L\) | 左移串行输入 | 左移时数据从此输入 |
| \(D_0 \sim D_3\) | 并行数据输入 | 置数模式时的 4 位输入 |
| \(Q_0 \sim Q_3\) | 并行输出 | 4 位输出 |

### 功能表

| \(\overline{CR}\) | \(S_1\) | \(S_0\) | \(CP\) | 功能 |
|:---:|:---:|:---:|:---:|:---|
| 0 | X | X | X | 异步清零（\(Q_0 \sim Q_3 = 0\)） |
| 1 | 0 | 0 | \(\uparrow\) | 保持 |
| 1 | 0 | 1 | \(\uparrow\) | 右移（\(D_R \to Q_0 \to Q_1 \to Q_2 \to Q_3\)） |
| 1 | 1 | 0 | \(\uparrow\) | 左移（\(D_L \to Q_3 \to Q_2 \to Q_1 \to Q_0\)） |
| 1 | 1 | 1 | \(\uparrow\) | 并行置数（\(D_i \to Q_i\)） |

### 典型应用

- **串并转换**：串行输入转并行输出（右移模式 + 并行读取）
- **并串转换**：并行置数后逐位移出（置数 + 右移模式）
- **序列检测**：配合逻辑门实现序列检测器
- **环形计数器**：将 \(Q_3\) 接 \(D_R\) 构成环形计数器

## B.5 74LS161 — 4 位同步二进制计数器

### 功能概述

74LS161 是 4 位同步二进制加法计数器（模 16），具有异步清零、同步置数、计数使能和进位输出功能，是计数器应用最广泛的芯片之一。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP\) | 时钟输入 | 上升沿触发 |
| \(\overline{CR}\) | 异步清零 | 低电平有效，立即清零 |
| \(\overline{LD}\) | 同步置数 | 低电平有效，下一 CP 加载 \(D_0 \sim D_3\) |
| \(EP, ET\) | 计数使能 | 均为高电平时计数 |
| \(D_0 \sim D_3\) | 并行数据输入 | 置数时载入的数据 |
| \(Q_0 \sim Q_3\) | 计数输出 | 当前计数值 |
| \(RCO\) | 进位输出 | \(Q_3 Q_2 Q_1 Q_0 = 1111\) 且 \(ET = 1\) 时为高 |

### 功能表

| \(\overline{CR}\) | \(\overline{LD}\) | \(EP\) | \(ET\) | \(CP\) | 功能 |
|:---:|:---:|:---:|:---:|:---:|:---|
| 0 | X | X | X | X | 异步清零 |
| 1 | 0 | X | X | \(\uparrow\) | 同步置数 |
| 1 | 1 | 1 | 1 | \(\uparrow\) | 计数（0000→0001→...→1111→0000） |
| 1 | 1 | 0 | X | X | 保持 |
| 1 | 1 | X | 0 | X | 保持（且 \(RCO = 0\)） |

### 典型应用

- **任意模计数器**：反馈清零法（\(\overline{CR}\)）或反馈置数法（\(\overline{LD}\)）实现模 \(N\) 计数器
- **级联扩展**：\(RCO\) 接下一片 \(ET\)，实现 8/12/16 位同步计数器
- **分频器**：模 \(N\) 计数器实现 \(N\) 分频

---

## B.6 74LS163 — 同步清零版

### 与 74LS161 的区别

74LS163 与 74LS161 功能几乎完全相同，唯一区别：**清零为同步方式**。

| 特性 | 74LS161 | 74LS163 |
|:---:|:---:|:---:|
| 清零方式 | **异步**清零 | **同步**清零 |
| 清零触发 | \(\overline{CR} = 0\) 立即清零 | \(\overline{CR} = 0\) 后需等 CP 上升沿才清零 |
| 反馈清零法 | 计数到 N 时产生清零，N 是瞬态 | 计数到 N-1 时产生清零，无瞬态 |

> 使用 74LS163 构成模 N 计数器时，反馈清零信号应在状态 \(N-1\) 产生（而非 \(N\)），因为清零动作要等到下一个 CP 才执行。

## B.7 74LS160 — 同步十进制计数器

### 功能概述

74LS160 是同步十进制加法计数器（模 10），引脚和功能与 74LS161 完全一致，区别在于计数循环为 0000~1001（十进制 0~9），到 1001 后返回 0000。

### 与 74LS161 的区别

| 特性 | 74LS161 | 74LS160 |
|:---:|:---:|:---:|
| 计数模 | 16（0000~1111） | 10（0000~1001） |
| \(RCO\) 触发 | \(Q = 1111\) 且 \(ET = 1\) | \(Q = 1001\) 且 \(ET = 1\) |
| 清零方式 | 异步清零 | 异步清零 |

### 典型应用

- **十进制计数/显示**：配合 7448 译码器和数码管实现十进制计数显示
- **分频器**：实现 10 分频
- **数字钟**：构成秒/分/时的十进制计数单元

---

## B.8 74LS191 — 同步可逆二进制计数器

### 功能概述

74LS191 是 4 位同步可逆二进制计数器（模 16），通过单时钟和加/减控制端实现加法或减法计数，带有进位/借位输出。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(CP\) | 时钟输入 | 上升沿触发 |
| \(\overline{LD}\) | 异步置数 | 低电平有效 |
| \(\overline{U}/D\) | 加减控制 | 0 = 加法计数，1 = 减法计数 |
| \(EN\) | 计数使能 | 低电平有效 |
| \(D_0 \sim D_3\) | 并行数据输入 | 置数时载入 |
| \(Q_0 \sim Q_3\) | 计数输出 | 当前计数值 |
| \(MAX/MIN\) | 最大/最小输出 | 加法到 1111 或减法到 0000 时为高 |

### 典型应用

- **可逆计数**：通过 \(\overline{U}/D\) 控制加/减方向
- **可编程分频器**：配合置数实现任意分频比

---

## B.9 74LS193 — 双时钟可逆二进制计数器

### 与 74LS191 的区别

| 特性 | 74LS191 | 74LS193 |
|:---:|:---:|:---:|
| 时钟 | 单时钟 | 双时钟（\(CP_U\) 加法，\(CP_D\) 减法） |
| 加减控制 | \(\overline{U}/D\) 引脚 | 由哪个时钟有脉冲决定 |
| 置数方式 | 异步置数 | 异步置数 |
| 清零 | 无独立清零端 | 异步清零（\(CLR\)） |

### 典型应用

- **双方向计数**：需要同时支持加减计数的场景
- **上下计数器**：位置/角度测量中的正反向计数

---

## B.10 74LS373 — 八 D 锁存器

### 功能概述

74LS373 是 8 位透明 D 锁存器，具有三态输出。当使能端有效时输出跟随输入（透明），使能端无效时锁存数据。广泛用于总线接口。

### 引脚说明

| 引脚 | 名称 | 功能 |
|:---:|:---|:---|
| \(D_0 \sim D_7\) | 数据输入 | 8 路并行数据 |
| \(Q_0 \sim Q_7\) | 数据输出 | 8 路并行输出（三态） |
| \(LE\) | 锁存使能 | 高电平透明，下降沿锁存 |
| \(\overline{OE}\) | 输出使能 | 低电平有效，高电平时输出高阻 |

### 功能表

| \(\overline{OE}\) | \(LE\) | \(D\) | \(Q\) |
|:---:|:---:|:---:|:---:|
| 1 | X | X | 高阻 |
| 0 | 1 | 0 | 0（透明） |
| 0 | 1 | 1 | 1（透明） |
| 0 | 0 | X | 锁存值 |

### 典型应用

- **地址锁存**：单片机系统中锁存低 8 位地址
- **总线缓冲**：三态输出实现总线隔离
- **数据暂存**：暂存 8 位并行数据
