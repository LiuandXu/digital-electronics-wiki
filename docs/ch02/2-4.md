# 2.4 逻辑函数化简

逻辑函数化简的目的是使数字电路**更简单、成本更低、功耗更低、速度更快**。化简的标准是：**与-或式中与项最少且每项的变量最少**，对应于电路就是所用门最少且每个门的输入端最少。

---

## 2.4.1 代数化简法

代数化简法是利用逻辑代数的基本公式和常用公式，通过代数运算消去多余乘积项和乘积项中多余因子。

### 1. 并项法

利用公式 \(AB + A\overline{B} = A\)，将两项合并为一项。

**例1**：\(Y_1 = A\overline{B}C + A\overline{B}\,\overline{C} = A\overline{B}(C + \overline{C}) = A\overline{B}\)

**例2**：\(Y_2 = \overline{A}BC + A\overline{B}C + AB\overline{C} + ABC\)（可两两合并）

### 2. 吸收法

利用公式 \(A + AB = A\)，消去多余项。

**例**：\(Y_3 = \overline{A}B + \overline{A}B\overline{C}D(E+F) = \overline{A}B\)

### 3. 消因子法

利用公式 \(A + \overline{A}B = A + B\)，消去多余因子。

**例**：\(Y_5 = AB + \overline{A}C + \overline{B}C = AB + C(\overline{A} + \overline{B}) = AB + C\overline{AB} = AB + C\)

### 4. 消项法

利用公式 \(AB + \overline{A}C + BC = AB + \overline{A}C\)，消去冗余项 \(BC\)。

**例1**：\(Y_6 = A\overline{B} + \overline{A}C + \overline{B}C = A\overline{B} + \overline{A}C\)

**例2**：\(Y_7 = AB + \overline{A}C + BCD = AB + \overline{A}C\)

### 5. 配项法

- 利用 \(A + A = A\) 添加重复项
- 利用 \(A + \overline{A} = 1\) 在某一项上乘以 \((A + \overline{A})\) 展开

**例**：\(Y_8 = AB + \overline{B}C + \overline{A}C + BC\overline{D}\)（配项后化简）

### 6. 综合法

复杂逻辑函数往往需要综合使用多种方法。

**例**：化简 \(Y = AB + \overline{A}C + \overline{B}C + \overline{A}B + \overline{A}CD\)

化简过程：
1. \(AB + \overline{A}B = B\)（并项法）
2. \(B + \overline{B}C = B + C\)（消因子法）
3. \(Y = B + C + \overline{A}C + \overline{A}CD = B + C + \overline{A}\)（吸收法）

### 变换为 "与非-与非" 形式

利用 De Morgan 定理进行变换：

\[
Y = AB + C + BC + BD = \overline{\overline{AB} \cdot \overline{BC} \cdot \overline{BD}}
\]

### 变换为 "或非-或非" 形式

\[
Y = A + BC = \overline{\overline{A + B} + \overline{C}}
\]

## 2.4.3 卡诺图化简法

卡诺图是1953年由贝尔实验室工程师**莫里斯·卡诺**发明的图形化化简方法，使逻辑函数化简变得简单直观。

### 1. 卡诺图的构成

- \(n\) 个变量对应 \(2^n\) 个小方格，每个小方格代表一个最小项
- 行列坐标按**格雷码**排列，保证几何相邻 = 逻辑相邻
- 卡诺图是一个**上下、左右闭合**的图形

**四变量卡诺图**：

| \(AB \setminus CD\) | 00 | 01 | 11 | 10 |
|:---:|:--:|:--:|:--:|:--:|
| **00** | \(m_0\) | \(m_1\) | \(m_3\) | \(m_2\) |
| **01** | \(m_4\) | \(m_5\) | \(m_7\) | \(m_6\) |
| **11** | \(m_{12}\) | \(m_{13}\) | \(m_{15}\) | \(m_{14}\) |
| **10** | \(m_8\) | \(m_9\) | \(m_{11}\) | \(m_{10}\) |

### 2. 卡诺图化简步骤

1. 将逻辑函数转换为**最小项之和**的标准形式
2. 画出函数的卡诺图（对应最小项填1）
3. **合并最小项**（画圈）
4. 每个圈写出一个乘积项
5. 所有乘积项相加，得到最简与或式

### 3. 画圈规则

1. 圈内的1必须是 \(2^n\) 个（1, 2, 4, 8, ...）
2. 圈要尽可能大，圈数尽可能少
3. 圈内所有1必须是逻辑相邻的
4. 不能出现没有新1的圈（避免冗余）
5. 1可以被多个圈重复使用
6. 最简结果不是唯一的

### 4. 化简示例

化简 \(F = AB\overline{C} + A\overline{C}D + \overline{A}BCD + ABC\)

① 化为最小项之和：填入卡诺图

② 画出卡诺图，标出所有为1的最小项

③ 画圈合并：

\[
F = A\overline{C}D + \overline{B} \cdot D + \overline{B} \cdot C
\]

!!! warning "易错点"
    画圈时注意卡诺图的**上下边界和左右边界也是相邻的**，可以跨边界画圈。另外，四角（\(m_0, m_2, m_8, m_{10}\)）也是相邻的！

## 2.4.4 具有无关项的逻辑函数化简

### 1. 无关项的定义

| 类型 | 定义 |
|------|------|
| **约束项** | 某些输入变量组合根本不可能出现，这些组合对应的最小项 |
| **任意项** | 某些输入组合出现时，输出是0还是1都无所谓，不影响电路功能 |

约束项和任意项统称为**无关项**，在卡诺图中用 "\(\times\)" 表示。

### 2. 利用无关项化简

化简原则：合并时可将 "\(\times\)" 当作 1 或 0，以能得到**最大的矩形组合**且**矩形数目最小**为准。

**示例**：化简 \(Y = A\overline{C}D + \overline{A}BCD + ABCD\)，已知约束条件

\[
ABCD + ABCD + ABCD + ABCD + ABCD + ABCD = 0
\]

将约束条件对应的无关项填入卡诺图（标 \(\times\)），合并时选择性利用，得到更简单的化简结果。

## 化简方法对比

| 方法 | 优点 | 缺点 |
|------|------|------|
| **代数法** | 适用于任何逻辑函数，灵活 | 无固定步骤，依赖经验；不易判断是否最简 |
| **卡诺图法** | 直观、有固定步骤，易判断是否最简 | 变量超过5~6个时操作困难 |
| **Q-M算法** | 适合计算机编程实现，适合多变量 | 手工操作繁琐 |

> **重点**：实际工程中，小规模逻辑（4个变量以内）优先用卡诺图；大规模逻辑由EDA综合工具自动完成优化。
