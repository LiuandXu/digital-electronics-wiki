## 关于本站

- 内容基于《数字电子技术基础》教材，结合工程实践整理而成
- 所有 Mermaid 图表、公式、代码均原创编写
- 支持**亮色 / 深色**模式自动切换，点击右上角图标切换
- 使用 `Ctrl + K` 快速搜索，`Ctrl + Shift + R` 强制刷新缓存

!!! tip "学习建议"
    1. 按章节顺序学习，每章先读正文再做总结
    2. 重点关注各章的「易错点」提示框
    3. 附录收录了常用 IC 芯片速查表，做题时随时查阅
    4. Verilog HDL 部分需要配合实际 FPGA 开发板练习
